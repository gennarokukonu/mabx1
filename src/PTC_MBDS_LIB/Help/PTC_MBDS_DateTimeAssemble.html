Date Time Assemble

  MATLAB Helpdesk 

General

Assembles a date and time signal from individual date and time component numbers, i.e. year, month, day, hours, minutes and seconds.

Library

MBDS Blockset

Description

    

The bock assembles a date and time signal with a data type date_time based on several date and time components provided as inputs. The inputs are year, month, day, hours, minutes and seconds representing a specific date and time value. The resulting date and time signal is limited to the range of 1970-01-01-00:00:00 to 2105-12-31-23:59:59. If the specified date and time value exceeds this interval, the output signal is automatically limited to the valid range.

The block has an ENO (Error Number Output) output port to indicate an error. The ENO output is set to false if the input data is invalid or out of range, otherwise it is set to true. The ENO output port can be hidden/shown via the parameter Show ENO port in the Main register card of the block mask.

Inputs and Outputs

Port I/O Data Type Description
Year
 In
 uint16 Year from 1970 to 2105
Month
 In uint8 Month from 1 to 12
Day In uint8 Day from 1 to 31
Hours In uint8 Hours from 0 to 23
Minutes In uint8 Minutes from 0 to 59
Seconds In uint8 Seconds from 0 to 59
DT Out uint32 Resulting date and time signal
ENO Out boolean ENO is false if an error has been occurred. If the block has been processed correctly the ENO Output is set to true.
