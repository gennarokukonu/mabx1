Modulo

 MATLAB Helpdesk 

General

Modulus after division.

Library

MBDS Blockset

Description

y = mod(u1,u2) if u2 ~= 0, returns u1 - n * u2 where n = floor(u1 / u2). By convention, mod(u1,0) is u1. The inputs u1 and u2 must be arrays of the same size or scalars.

Inputs and Outputs

Port I/O Data Type Description
u1 In Any unsigned integer First input signal
u2 In Any unsigned integer Second input signal
y Out Any unsigned integer Output signal
