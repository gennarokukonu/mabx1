Scroll Line No Wrap

 MATLAB Helpdesk 

General

Scroll within a list box and provide a selected/highlighted item. There is no roundtrip or wrap around at the beginning or end of the list.

Library

MBDS Blockset

Description

   

This block is used to scroll line by line within a list boxes, check boxes, and page containers. The main difference to block ScrollCenteredNoWrap is that the selected/highlighted item is not located as centered line in general. Scrolling is only performed if it is required (the selected/highlighted item would move out of the visible range).

A visualization of the scrolling behavior using this block is illustrated in the following figure:

   
   Figure 1: List box with nItems = 6 and nVisItens = 3 items

Remarks:
  • Keep in mind that item i has index i -1.
  • As distinguished from block ScrollLineRoundTrip and ScrollLineWrapAround, there is no roundtrip or wrap around at the beginning or end of the list box.
    • No scrolling up for SelItem = 0.
    • No scrolling down for SelItem = nItems -1.
  • Scrolling up/down referes to the visual location and not to the item number.
    • Scrolling up from SelItem = 4 leads to SelItem = 3.
    • Scrolling down from SelItem = 4 leads to SelItem = 5.
  • The first item is selected/highlighted initially.

Inputs and Outputs

Port I/O Data Type Description
Up In boolean Signal to move up the selected item
Down In boolean Singal to move down the selected item
nItems In uint8 Total number of list box items
nVisItems In uint8 Number of visible items
FirstItem Out uint8 Index of the first visible item
SelItem Out uint8 Index of the selected item
ItemAbove Out boolean Indicates if further items exist above the selected item
ItemBelow Out boolean Indicates if further items exist below the selected item
