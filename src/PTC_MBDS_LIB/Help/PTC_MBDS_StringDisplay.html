String Display

 MATLAB Helpdesk 

General

Displays a string signal as text on the block. 

Library

MBDS Blockset

Description

    

This block displays an uint8 array signal connected to the input port as ASCII text on the block mask. If the string contains special characters for linefeed or carriage return the string is split in several rows. The number of displayed rows can be limited via the parameter Maximum number of rows displayed in the block mask. 

Inputs and Outputs

Port I/O Data Type Description
String
 In
 uint8 (N) String signal to be displayed
