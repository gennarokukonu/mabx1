String Find

  MATLAB Helpdesk 

General

Search a partial string in another string.

Library

MBDS Blockset

Description

    

A partial string connected to In2 is searched in the string defined at input In1. The output returns the position where the first match of both strings occurs. If the partial string cannot be found the output is set to 0.
 
The block has an ENO (Error Number Output) output port to indicate an error. The ENO output is set to false in case of an error, otherwise it is set to true. The ENO output port can be hidden/shown via the parameter Show ENO port in the block mask.

Inputs and Outputs

Port I/O Data Type Description
In1
 In
 uint8 (N) String to search inside.
In2
 In uint8 (N) String to be found in the string connected to In1
Out Out int32 The position at which the first match of both strings occurs.
ENO Out boolean ENO (Error Number Output) is false (0) if an error has been occurred. If the block has been processed correctly the ENO Output is set to true (1).
