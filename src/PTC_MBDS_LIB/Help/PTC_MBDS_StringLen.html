MBDS Environment
 

String Len

  MATLAB Helpdesk 

General

Get the length of the string connected to input.

Library

MBDS Blockset

Description

    

The length of the string connected to the input is determined, i.e. the number of ASCII characters (without string termination characters) are returned.

Inputs and Outputs

Port I/O Data Type Description
In
 In
 uint8 (N) Input string
Out Out int16 String length of the connected string
