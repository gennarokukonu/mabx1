MBDS Environment
 

String Right

  MATLAB Helpdesk 

General

Get a specified number of the most right characters of a string.

Library

MBDS Blockset

Description

    

The specified number of the most right characters are output by the block. The number of characters is specified by input L. For different values of L the block behaves as follows:
  • If L exceeds the string length of string In1, the string is output without changes, but the ENO output is set to false.
  • If L is zero an empty string is output and the ENO output is set to true.
  • If L is negative an empty string is output and the ENO output is set to false.
The block has an ENO (Error Number Output) output port to indicate an error. The ENO output is set to false in case of an error, otherwise it is set to true. The ENO output port can be hidden/shown via the parameter Show ENO port in the block mask.

Inputs and Outputs

Port I/O Data Type Description
In
 In
 uint8 (N) Connected String
L
 In Any integer Number of characters (L) are taken from the entered string, starting at the right.
Out Out uint8 (N) Output string
ENO Out boolean
 ENO is false if an error has been occurred. If the block has been processed correctly the ENO Output is set to true.
Example

Examples for illustration: String 'abcde' is connected to input In.

Example 1:

  • L = 3 : 3 characters are taken from the string, starting at the right.
  • Result: 'cde'

Example 2:

  • L =  7 : 7 characters would be taken from the string, starting at the right. However, as the string only contains 5 characters, the string is output without any changes.
  • Result: 'abcde'

Example 3:

  • L: -2 = The negative input causes an empty output.
  • Result: ''

For more information refer to the example.
