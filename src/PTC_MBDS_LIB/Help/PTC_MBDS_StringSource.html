 MATLAB Helpdesk 

General

Generates a constant string signal.

Library

MBDS Blockset

Description

    

This block outputs an uint8 array based on the ASCII codes of the string specified by the user. Specify the string via the parameter Constant value of the block mask. The string is automatically limited to the maximum string length specified in the Project Options.

Inputs and Outputs

Port I/O Data Type Description
Out
 Out
uint8 (N) Constant string signal
