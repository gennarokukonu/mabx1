MBDS Environment
 

Toggle

 MATLAB Helpdesk 

General

Outputs a logical value which can be toggled between true and false by double click on the block. 

Library

MBDS Blockset

Description

      

The block acts like a toggle button having only two possible values. The value of the block can be toggled by double click on the block. The block provides an output 1 when it is in true state and 0 when it is in false state.

Inputs and Outputs

Port I/O Data Type Description
Out Out boolean Output is 1 when true and 0 when in false state
