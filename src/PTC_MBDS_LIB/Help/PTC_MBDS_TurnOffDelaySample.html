TurnOffDelaySample

 MATLAB Helpdesk 

General

Falling edge integer delay.

Library

MBDS Blockset

Description

A falling edge of the input signal u is delayed by n block evaluations (E).

Using the parameter mask the block can be configured according to the users needs.Via register card Main the block input and output ports can be shown or hidden (see Figure 1).

  • Show R as Inport. If checked, the block provides a reset input port. If unchecked, the block cannot be reset.


Figure 1: Parameter Mask Dialog - Register card Block Main

The register card Version displays the block version information.

Inputs and Outputs

Port I/O Data Type Description
n
 In Any* Number of block evaluations for delaying
u
 In boolean Input signal
R In boolean Resets the block state to its initial value while true
y Out boolean Delayed (n-block evaluations) falling edge of the input signal
*) Except enumeration data types.
