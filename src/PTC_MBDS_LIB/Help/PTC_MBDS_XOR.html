XOR

 MATLAB Helpdesk 

General

Logical or bitwise XOR operator. 

Library

MBDS Blockset

Description 


Logical or bitwise XOR operator. For multiple inputs, operators are applied across the inputs. For a single input, operators are applied across the input vector. In bitwise mode the bit mask can be applied onto the single input signal.

Using the parameter mask the block can be configured according to the users needs. Via register card Main the following options are available (see Figure 1).
  • Use 'bitwise' mode. If checked  the block provides a bitwise XOR operation onto the inputs.
  • Number of input ports: This field can be used to define the number of input ports.
  • Use bit mask.  This field is enabled in bitwise mode. If checked the parameter Number of input ports is fixed to 1 and the bit mask parameter is applied onto the signal. 
  • Bit mask. This field is enabled if parameter Use bit mask is checked. The bitmask can be specified by a numeric value (decimal) or a string (hexadecimal), e.g. 'E2A7'.


Figure 1: Parameter Mask Dialog - Register card Main

Via the register card Signal Attributes signal related attributes can be configured (see Figure 2).

  • Output data type. The data type of the output signal can be configured here, if bitwise mode is selected.


Figure 2: Parameter Mask Dialog - Register card  Signal Attributes

The register card Version displays the block version information.
